# 🚀 Smart Predictive Maintenance System

An AI-powered Predictive Maintenance System that analyzes industrial machine sensor data to predict maintenance requirements, identify potential failure types, estimate the Remaining Useful Life (RUL), and provide intelligent maintenance recommendations.

The project combines Machine Learning models with an interactive dashboard to help reduce unexpected machine downtime and improve maintenance planning.

---

## 📌 Project Overview

This project uses Industrial IoT sensor data to predict:

- ✅ Maintenance Required (Classification)
- ⚙️ Failure Type (Multi-class Classification)
- ⏳ Remaining Useful Life (Regression)
- 📊 Risk Factors Detection
- 💡 Rule-Based Maintenance Recommendations

- SVM
